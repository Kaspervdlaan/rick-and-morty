# Rick and Morty Project

A simple project to explore the Rick and Morty universe.

## How to start project

1. **Install dependencies:**
  ``` 
  npm install
  ```

2. **Start the development server:**
  ```
  npm run dev
  ```

## Dependencies

- [React]
- [Axios]
- [React Router]
- [React Icons]
- [Tailwind]
